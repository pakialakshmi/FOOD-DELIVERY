Food Delivery Sales Analysis

Project Overview

This project analyzes food delivery order data to understand customer ordering patterns, identify peak order hours, determine the most popular dishes, and analyze revenue generated by different food items.

Objectives

- Analyze customer order data
- Identify peak order times
- Find the most popular food items
- Calculate revenue generated by each dish
- Analyze orders based on location
- Generate business insights using Python Data Analytics

Tools & Technologies

- Python
- Pandas
- Matplotlib
- Google Colab
- GitHub

Dataset

A sample dataset containing:

- Order ID
- Order Time
- Location
- Item
- Amount

Features

- Dataset Preview
- Peak Order Hour Analysis
- Popular Dish Analysis
- Revenue Analysis
- Location-wise Order Analysis
- Business Insights Dashboard
- Order Filtering by Item and Location

Outcome

The project helps identify customer preferences, busiest ordering hours, top-selling dishes, and revenue trends, enabling better business decision-making for food delivery services.
